A bit about me

Hey — I'm Sphumelele, a final-year Diploma in Software Development student based in Durban. I've spent the past three years studying and getting the know all on how software works, and how apps and websites are built and what is needed to make this function smoothly. I have a passion for creating clean, efficient code and building things that people find useful and enjoyable to use. I do have more an obsession for clean looking interfaces and intuitive user experiences. I believe that good design is just as important as good code, and I strive to create things that are not only functional but also a pleasure to use.

Right now I'm focused on full-stack web development, specifically working with C++, HTML, and MySQL, PL/SQL. I care about code that's readable, systems that are honest about their complexity, and interfaces that don't need a tutorial.

Outside of lectures I'm either contributing to open-source projects, building random side things, or convincing friends that dark mode isn't a personality trait.
